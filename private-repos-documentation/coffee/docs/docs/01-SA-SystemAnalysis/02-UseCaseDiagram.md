## 5. 詳細使用案例：POS 點餐流程

```mermaid
sequenceDiagram
    participant 店員
    participant POS系統
    participant 商品服務
    participant 客戶服務
    participant 訂單服務
    participant 會員服務
    
    店員->>POS系統: 開始新訂單
    店員->>POS系統: 選擇商品
    POS系統->>商品服務: 取得商品資訊
    商品服務-->>POS系統: 返回商品資料
    
    opt 會員訂單
        店員->>POS系統: 輸入會員資訊
        POS系統->>客戶服務: 查詢會員
        客戶服務-->>POS系統: 返回會員資料
        POS系統->>會員服務: 查詢可用點數
        會員服務-->>POS系統: 返回點數資訊
    end
    
    店員->>POS系統: 確認訂單
    POS系統->>POS系統: 計算總金額
    店員->>POS系統: 選擇付款方式
    POS系統->>訂單服務: 建立訂單
    訂單服務-->>POS系統: 返回訂單編號
    
    opt 會員累積點數
        POS系統->>會員服務: 累積點數
        會員服務-->>POS系統: 點數累積成功
    end
    
    POS系統-->>店員: 顯示訂單完成
    opt 列印發票
        店員->>POS系統: 列印發票
        POS系統-->>店員: 發票列印完成
    end
```

## 6. 詳細使用案例：採購入庫流程

```mermaid
sequenceDiagram
    participant 採購人員
    participant ERP系統
    participant 採購服務
    participant 庫存服務
    participant 通知服務
    
    採購人員->>ERP系統: 建立採購單
    ERP系統->>採購服務: 提交採購單
    採購服務-->>ERP系統: 返回採購單號
    
    採購人員->>ERP系統: 提交審核
    ERP系統->>通知服務: 發送審核通知
    
    alt 審核通過
        採購主管->>ERP系統: 審核通過
        ERP系統->>採購服務: 更新狀態為已審核
        ERP系統->>通知服務: 通知採購人員
    else 審核拒絕
        採購主管->>ERP系統: 審核拒絕
        ERP系統->>採購服務: 更新狀態為已拒絕
        ERP系統->>通知服務: 通知採購人員
    end
    
    採購人員->>ERP系統: 進貨入庫
    ERP系統->>採購服務: 記錄入庫
    採購服務->>庫存服務: 更新庫存數量
    庫存服務-->>採購服務: 庫存更新成功
    採購服務-->>ERP系統: 入庫完成
    ERP系統-->>採購人員: 顯示完成訊息
```

## 9. 使用案例關聯說明

### Include 關係
- **訂單結帳** includes **查詢會員** - 結帳時必須查詢會員資訊
- **進貨入庫** includes **更新庫存** - 入庫必定更新庫存
- **新增用戶** includes **分配角色** - 新增用戶必須分配角色

### Extend 關係
- **訂單結帳** extends **列印發票** - 可選擇是否列印發票
- **庫存預警** extends **建立採購單** - 低於安全庫存時可自動建議採購

### Generalization 關係
- **查詢報表** 是所有報表類使用案例的泛化
- **管理主檔** 是所有管理類使用案例的泛化
