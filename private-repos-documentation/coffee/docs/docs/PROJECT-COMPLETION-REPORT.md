## 3. 技術架構概覽

### 3.1 微服務架構 (8個服務)
```
API Gateway (Gin + Go)
├── Auth Service          # 使用者認證與授權
├── Customers Service     # 客戶資料管理 (CRM)
├── Orders Service        # 訂單管理 (POS)
├── Inventory Service     # 庫存管理 (ERP)
├── Loyalty Service       # 會員忠誠度管理
├── Notifications Service # 通知服務 (多渠道)
└── Reports Service       # 報表分析 (BI)
```

### 3.2 技術棧規格
```yaml
後端:
  語言: Go 1.21+
  框架: Gin 1.9+
  資料庫: MySQL 8.0+ (32 張表)
  快取: Redis 7.0+
  消息隊列: NATS 2.9+

前端:
  語言: TypeScript 5.0+
  框架: Vue 3.3+ / Nuxt 3.7+
  UI 框架: Element Plus / Tailwind CSS

基礎設施:
  容器化: Docker + Kubernetes
  監控: Prometheus + Grafana
  CI/CD: GitHub Actions
  文檔: OpenAPI 3.0
```

### 3.3 資料庫設計
- **32 張資料表**完整設計
- **ACID 事務**保證資料一致性
- **索引優化**確保查詢效能
- **主從複寫**支援高可用架構
