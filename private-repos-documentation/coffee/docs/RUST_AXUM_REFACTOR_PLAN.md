## 🛠️ **技術實作要求**

### Rust技術棧
- **Framework**: Axum (async web framework)
- **資料庫**: sqlx (PostgreSQL連接)
- **序列化**: serde (JSON處理)
- **認證**: jsonwebtoken (JWT處理)
- **密碼**: bcrypt (密碼雜湊)
- **Redis**: redis-rs (Redis客戶端)
- **測試**: tokio-test (異步測試)

### 標準專案結構
```
services/{service_name}/
├── src/
│   ├── main.rs
│   ├── handlers/           # API處理器
│   ├── models/            # 資料模型
│   ├── services/          # 業務邏輯
│   ├── db/               # 資料庫操作
│   └── utils/            # 工具函數
├── Cargo.toml
├── Dockerfile
└── tests/                # 單元測試
```

### 共用Dependencies
```toml
[dependencies]
axum = "0.7"
tokio = { version = "1.0", features = ["full"] }
```
