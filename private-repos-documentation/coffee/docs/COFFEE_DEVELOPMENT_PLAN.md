### 🎯 **系統概述**
**系統類型**: 連鎖咖啡館管理系統 (CRM + ERP雙系統)
**核心功能**:
- **CRM (門店系統)**: POS點餐、客戶管理、會員服務、門店報表
- **ERP (總部系統)**: 營運總覽、門店管理、庫存管理、財務報表、供應商管理

**技術架構**:
- **後端**: Go + Gin微服務架構 (8個微服務)
- **前端**: Vue 3 + Nuxt 3 + TypeScript
- **資料庫**: MySQL 8.0 + Redis快取
- **容器**: Docker + Kubernetes部署
