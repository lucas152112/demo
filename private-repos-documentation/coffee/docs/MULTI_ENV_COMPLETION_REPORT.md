## 🏗️ **多環境架構設計**

### 環境分離策略
```
DEV 環境    → coffee_dev_*   資料表前綴
TEST 環境   → coffee_test_*  資料表前綴
PP 環境     → coffee_pp_*    資料表前綴
PROD 環境   → coffee_prod_*  資料表前綴
```

### K8s 資料庫服務配置
- **MySQL 8.0**: mysql-service.database.svc.cluster.local:3306/gamedb
- **Redis 7.0**: redis-service.database.svc.cluster.local:6379 (Key前綴隔離)
- **MongoDB 7.0**: mongodb-service.database.svc.cluster.local:27017 (Database隔離)
