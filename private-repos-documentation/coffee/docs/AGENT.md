## 🔧 Agent 配置與設定

### 程式碼規範偏好
```yaml
Backend (Go):
  - Framework: Gin
  - Error Handling: 統一錯誤處理機制
  - Naming: camelCase for variables, PascalCase for types
  - Structure: Clean Architecture pattern

Frontend (Vue):
  - Framework: Vue 3 + Nuxt 3
  - Language: TypeScript
  - Styling: Tailwind CSS (建議)
  - State Management: Pinia

Database:
  - RDBMS: MySQL 8.0+
  - Naming: snake_case
  - Indexing: 查詢最佳化導向
```

### 文件撰寫風格
- **語言**: 繁體中文為主，技術術語保留英文
- **格式**: Markdown 格式，結構清晰
- **圖表**: Mermaid 語法繪製流程圖
- **範例**: 提供實際程式碼範例

### 回應模式
- **主動建議**: 發現潛在問題時主動提醒
- **詳細解釋**: 提供技術決策的原因與考量
- **實用導向**: 著重可執行的具體建議
- **持續改進**: 根據專案進展調整建議
