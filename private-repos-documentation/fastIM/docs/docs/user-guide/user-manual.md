## 🚀 快速開始

### 註冊和登入

#### 1. 首次使用
1. 下載對應平台的 FastIM 客戶端
2. 安裝並啟動應用程式
3. 點擊「註冊新帳號」
4. 填寫用戶資訊：
   - 用戶名（3-20字符，字母數字下劃線）
   - 密碼（至少8字符，包含大小寫字母和數字）
   - 電子郵件（用於找回密碼）
   - 顯示名稱（可選）

#### 2. 登入
1. 啟動 FastIM
2. 輸入用戶名和密碼
3. 選擇「記住我」（可選）
4. 點擊「登入」

#### 3. 首次配置
登入後系統會引導您：
1. 選擇頭像（可上傳自定義圖片）
2. 設置狀態（在線/忙碌/離開/勿擾）
3. 配置通知偏好
4. 加入或創建頻道

## 💻 客戶端使用

### Web 客戶端

#### 系統要求
- 現代瀏覽器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- 穩定的網路連接
- JavaScript 已啟用

#### 訪問方式
1. 直接訪問：`https://fastim.yourdomain.com`
2. 輸入登入憑證
3. 開始聊天

#### Web 功能特色
- 📱 響應式設計，適配各種屏幕尺寸
- 🔄 自動同步消息
- 📋 支持拖拽文件上傳
- ⌨️ 豐富的鍵盤快捷鍵
- 🌙 深色/淺色主題切換

### Flutter 桌面客戶端

#### 安裝方式

**Linux**
```bash
# 使用 deb 包
sudo dpkg -i fastim_1.0.0_arm64.deb

# 或從源碼安裝
git clone <fastim-repo>
cd fastim/frontend/flutter
flutter build linux --release
```

**Windows**
```powershell
# 下載 MSIX 安裝包
# fastim-1.0.0.msix
# 雙擊安裝或使用 PowerShell
Add-AppxPackage fastim-1.0.0.msix
```

**macOS**
```bash
# 下載 DMG 文件
# FastIM-1.0.0.dmg
# 雙擊掛載，拖拽到應用程式目錄
```

#### 桌面客戶端特色
- 🚀 原生性能，啟動快速
- 💾 本地緩存，離線查看歷史消息
- 🔔 系統原生通知
- ⚡ 多窗口支持
- 📁 文件拖拽發送

## 📢 基本功能

### 消息發送

#### 文字消息
1. 在消息輸入框中輸入文字
2. 按 Enter 發送（Shift+Enter 換行）
3. 支持 Markdown 格式：
   - **粗體文字**
   - *斜體文字*
   - `程式碼`
   - [連結](https://example.com)
   - > 引用文字

#### 表情符號
- 輸入 `:smile:` 自動轉換為 😊
- 點擊表情按鈕選擇
- 支持自定義表情包

#### 文件分享
1. **拖拽上傳**：直接拖拽文件到聊天窗口
2. **點擊上傳**：點擊附件按鈕選擇文件
3. **支持格式**：
   - 圖片：JPG, PNG, GIF, WebP (最大 10MB)
   - 文檔：PDF, DOC, DOCX, TXT (最大 50MB)
   - 其他：ZIP, RAR (最大 100MB)

#### 語音消息（移動端）
1. 長按麥克風按鈕
2. 開始錄音（最長 60 秒）
3. 鬆開發送，上滑取消

### 頻道管理

#### 加入頻道
1. 點擊「加入頻道」按鈕
2. 輸入頻道名稱或邀請碼
3. 等待管理員批准（如需要）

#### 創建頻道
1. 點擊「創建頻道」
2. 填寫頻道資訊：
   - 頻道名稱
   - 描述
   - 隱私設置（公開/私有）
   - 邀請權限
3. 點擊「創建」

#### 頻道設置
- **通知設置**：全部/僅提及/靜音
- **角色權限**：管理員/成員/觀察者
- **消息歷史**：保留期限設置

### 個人設置

#### 個人資料
- **頭像**：支持上傳 JPG/PNG，自動裁剪
- **顯示名稱**：在聊天中顯示的名稱
- **狀態消息**：自定義狀態文字
- **在線狀態**：在線/忙碌/離開/勿擾/隱身

#### 隱私設置
- **消息已讀回執**：開啟/關閉
- **最後上線時間**：顯示/隱藏
- **個人資料可見性**：公開/僅好友/私有

#### 通知設置
- **桌面通知**：啟用/禁用
- **聲音提醒**：選擇提示音
- **通知時間**：設置勿擾時段
- **關鍵詞提醒**：自定義關鍵詞

## ⚡ 高級功能

### 消息搜索

#### 全局搜索
1. 點擊搜索圖標或按 `Ctrl+F`
2. 輸入搜索關鍵詞
3. 支持的搜索語法：
   - `from:username` - 搜索指定用戶的消息
   - `in:channel` - 搜索指定頻道
   - `has:file` - 搜索包含文件的消息
   - `before:2024-01-01` - 搜索指定日期前
   - `after:2024-01-01` - 搜索指定日期後

#### 高級過濾
- **消息類型**：文字/圖片/文件/語音
- **時間範圍**：今天/本週/本月/自定義
- **用戶過濾**：特定用戶或排除用戶
- **頻道範圍**：全部/當前/指定頻道

### 消息操作

#### 消息回覆
1. 右鍵點擊消息
2. 選擇「回覆」
3. 輸入回覆內容
4. 發送後會顯示引用關係

#### 消息編輯
1. 右鍵點擊自己的消息
2. 選擇「編輯」
3. 修改內容
4. 按 Enter 保存（5分鐘內可編輯）

#### 消息刪除
1. 右鍵點擊消息
2. 選擇「刪除」
3. 確認操作
4. 支持批量刪除（管理員）

#### 消息固定
1. 右鍵點擊重要消息
2. 選擇「固定消息」
3. 固定的消息會顯示在頻道頂部
4. 每個頻道最多固定 10 條消息

### 文件管理

#### 文件預覽
- **圖片**：點擊縮略圖查看大圖
- **文檔**：支持 PDF 在線預覽
- **代碼**：語法高亮顯示
- **壓縮包**：顯示文件列表

#### 文件下載
1. 點擊文件名
2. 選擇保存位置
3. 支持斷點續傳
4. 下載進度實時顯示

#### 雲端存儲
- 自動上傳到雲端存儲
- 文件去重，節省空間
- 支持文件分享連結
- 設置文件過期時間

### 群組通話（計劃功能）

#### 語音通話
1. 在頻道中點擊「語音通話」
2. 等待其他用戶加入
3. 支持靜音/取消靜音
4. 支持屏幕共享

#### 視頻會議
1. 發起視頻通話邀請
2. 最多支持 10 人同時在線
3. 支持會議錄制
4. 共享屏幕和應用程式

## 🛠️ 故障排除

### 常見問題

#### 連接問題
**問題**：無法連接到服務器
**解決方案**：
1. 檢查網路連接
2. 確認服務器地址正確
3. 檢查防火牆設置
4. 嘗試使用不同的網路

**問題**：經常斷線重連
**解決方案**：
1. 檢查網路穩定性
2. 調整心跳間隔設置
3. 更新客戶端版本
4. 聯繫網路服務提供商

#### 消息問題
**問題**：消息發送失敗
**解決方案**：
1. 檢查網路連接
2. 確認用戶權限
3. 檢查消息內容是否被過濾
4. 嘗試重新發送

**問題**：消息不同步
**解決方案**：
1. 手動刷新頻道
2. 重新登入帳號
3. 清理本地緩存
4. 檢查時間同步

#### 文件問題
**問題**：文件上傳失敗
**解決方案**：
1. 檢查文件大小限制
2. 確認文件格式支持
3. 檢查存儲空間
4. 嘗試壓縮文件

**問題**：文件下載慢
**解決方案**：
1. 檢查網路速度
2. 嘗試在其他時間下載
3. 使用有線網路
4. 聯繫管理員

#### 性能問題
**問題**：應用程式響應慢
**解決方案**：
1. 關閉不必要的程式
2. 清理系統垃圾
3. 重啟應用程式
4. 檢查系統資源

**問題**：記憶體使用高
**解決方案**：
1. 清理聊天記錄緩存
2. 減少同時打開的頻道
3. 關閉不需要的功能
4. 重啟應用程式

## ❓ 常見問題

### 帳號相關

**Q: 忘記密碼怎麼辦？**
A: 1. 點擊登入頁面的「忘記密碼」
   2. 輸入註冊時的郵箱地址
   3. 查收重置密碼郵件
   4. 按照郵件指引重置密碼

**Q: 如何更改用戶名？**
A: 用戶名創建後無法更改，但可以修改顯示名稱：
   1. 進入個人設置
   2. 修改「顯示名稱」欄位
   3. 保存更改

**Q: 如何刪除帳號？**
A: 1. 進入設置 → 帳號安全
   2. 點擊「刪除帳號」
   3. 確認操作並輸入密碼
   4. 帳號將在 7 天後永久刪除

### 功能使用

**Q: 如何創建私人頻道？**
A: 1. 點擊「創建頻道」
   2. 選擇「私有頻道」
   3. 設置邀請權限
   4. 邀請特定用戶加入

**Q: 可以同時在多個設備登入嗎？**
A: 可以，FastIM 支持多設備同時在線，消息會自動同步到所有設備。

**Q: 如何備份聊天記錄？**
A: 1. 進入設置 → 數據管理
   2. 選擇「備份聊天記錄」
   3. 選擇備份範圍和格式
   4. 下載備份文件

### 技術支持

**Q: 支持哪些作業系統？**
A: - 桌面端：Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
   - 移動端：Android 8.0+, iOS 13.0+
   - Web：Chrome 90+, Firefox 88+, Safari 14+

**Q: 是否開源？**
A: FastIM 部分組件開源，詳情請查看 GitHub 倉庫。

**Q: 如何報告 Bug？**
A: 1. 前往 GitHub Issues 頁面
   2. 搜索是否已有相似問題
   3. 創建新 Issue，提供詳細信息
   4. 包含日誌文件和截圖

**Q: 如何獲取技術支持？**
A: - 郵箱：support@fastim.com
   - GitHub Issues
   - 社區論壇
   - 在線客服（工作時間：9:00-18:00）
